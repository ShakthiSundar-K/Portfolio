Quiz Maker Appliction

The Quiz Maker Application is a MERN stack-based project that enables users to create, manage, and participate in quizzes with advanced features. It supports full CRUD operations, allowing users to create both public and private quizzes, with private quiz IDs being sent via email for restricted access. Participants can take quizzes with a timer option, auto-next navigation, and auto-submit on timeout, while the system automatically evaluates scores upon submission. A leaderboard tracks top performers, and quiz analytics provide creators with detailed insights into participant performance and engagement.

Student Mentor Matching System

This project is a **Node.js and MongoDB-based backend API** designed for managing mentor-student relationships efficiently. It provides functionalities to create mentors and students, assign one or multiple students to a mentor, and reassign a new mentor to a student when needed. Additionally, the API enables tracking a student's previously assigned mentors and retrieving all students assigned to a particular mentor. As this is purely a backend project, it focuses on handling data and business logic without a frontend interface.
